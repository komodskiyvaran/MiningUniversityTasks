Практическое занятие №7.
Тема: Проектирование и реализация интерфейсов

Задание:

Спроектировать структуру классов в соответствии с
индивидуальным вариантом (см. приложение), минимальными средствами,
таким образом, чтобы метод(ы) добавленного в структуру классов
интерфейса участвовал(и) в определении вычисляемого показателя.
Объекты создавать массивами, инициализируя свойства случайным образом.

Базовый класс: Предприятие малого бизнеса

Производные классы:  Магазин, парикмахерская

Вычисляемый показатель: Название предприятия с максимальным числом сотрудников
