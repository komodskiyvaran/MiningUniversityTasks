Практическое занятие №10.
Тема: Программирование и исследование
процесса формирования списков на базе коллекций
ArrayList и LinkedList

В соответствии с индивидуальным заданием (см. приложение) разработать базовый класс и классы-наследники. Сформировать линейный список из 100 000 объектов производных классов.

Разработать меню, обеспечивающее:
1. создание списка;
2. поиск элемента списка по значению параметра;
3. удаление объекта из списка с заданным параметром;
4. расчет Вычисляемого показателя;
5. выход из программы.

Вычислить время выполнения каждой операции со списком при использовании коллекции ArrayList и коллекции LinkedList.

Базовый класс: Предприятие малого бизнеса

Производные классы:  Магазин, парикмахерская

Вычисляемый показатель: Название предприятия с максимальным числом сотрудников
